样式结构说明

1.样式结构

1.层级结构

  1. base层：样式重置，原子类样式；
  2. layout层：页面结构布局，样式类以'l-*'命名，此处只放页面结构框架类；
  3. module层：模块布局，样式类以'm-*'命名，此处只放模块内样式类；
  4. state层：状态层，样式可以选项方式'(s-)*'命名，主要指具体某个控件类；
  5. style层：主题层，需要重置的颜色，字体等。

2.基本规范

样式全部采用类标注，由于ID优先级较高，没有复用性，样式重置也比较困难，所以请不要采用ID样式。

样式类全部采用小写，多个单词请用中划线连接。个性化类跟在公共类的最后端。

在前三层中尽量不要采用子选择器超过三层的定义，特别是公共样式，这样会引起个性化的样式定制优先级不高。并且，公共样式慎用'!important',这样也会引起个性化样式定制优先级不高。

3.典型样式规则

1.行内标签不能写在块状标签之内

<span> //类似这种的写法很糟糕
    <div>
        ...
    </div>
</span>

2.全局不能使用绝对定位

3.块状标签居中

.block-center
{
    width:960px;
    margin:auto;
}

4.行内标签包括文字居中

.block-center
{
    text-align:center;  
}

5.行内标签宽度和高度

.inline-element
{
    display:inline-block; //以行内块显示宽度和高度才会生效
    width:100px;
    height:100px;
}

6.边框，分割线

模块间的分割线、盒子的边框统一采用border，不要采用hr,或者<div class="line"> </div>

.top-border
{
    border-top:1px solid red;
}

.red-border
{
    border:1px solid red;
}

2.表单

表单以全局控件方式提供全局样式。form全局样式类'form',input全局样式类'from-input'，select全局样式类'form-select'，textarea全局样式类'form-text' 。

默认所有'form-*'元素的宽度属性为'width:100%；'。label和前面提到的控件包裹在.form-field中可以提供良好的排列。

1.默认表单

默认表单label和内容都渲染为block级别的控件，各占一行。

<form class="form">
    <div class="form-field">
        <label><span class="required">*<span>name:</label>
        <input class="form-input"/>
    </div>
    <div class="form-field">
        <label><span class="required">*<span>school:</label>
        <select class="form-select"></select>
    </div>
    <div class="form-field">
        <label><span class="required">*<span>content:</label>
        <textarea class="form-text"></textarea>
    </div>
    <div class="form-field">
        <button>Cancel</button>
        <button>Save</button>
    </div>
</form>

2.水平排列的表单

为<form> 元素添加 .form-hr(horizontal) 类可使label和内容都渲染为 inline-block 级别的控件。

<form class="form form-hr">
    .....
</form>

可能需要手动设置宽度

在内联表单，我们将这些'form-*'元素的宽度设置为 'width:255px;'(将要重构为width:auto;)或其他，因此，多个控件可以排列在同一行。根据你的布局需求，可能需要一些额外的定制化需求宽度等。

3.左右两列的表单

<form class="form">
    <div class="form-left form-col-2">
        <div class="form-filed">
            ...
        </div>
    </div>
    <div class="form-right form-col-2">
        <div class="form-filed">
            ...
        </div>
    </div>
</form>
